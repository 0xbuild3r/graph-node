Module graph::runtime[−][src]

Facilities for creating and reading objects on the memory of an AssemblyScript (Asc) WASM module. Objects are passed through the asc_new and asc_get methods of an AscHeap implementation. These methods take types that implement To/FromAscObj and are therefore convertible to/from an AscType.
