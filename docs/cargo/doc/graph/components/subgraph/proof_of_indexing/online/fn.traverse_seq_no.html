fn traverse_seq_no(counts: &[usize]) -> Blake3SeqNo

Go directly to a SequenceNumber identifying a field within a struct. This is best understood by example. Consider the struct:

struct Outer { inners: Vec, outer_num: i32 } struct Inner { inner_num: i32, inner_str: String, }

Let’s say that we have the following data: Outer { inners: vec![ Inner { inner_num: 10, inner_str: “THIS”, }, ], outer_num: 0, }

And we need to identify the string “THIS”, at outer.inners[0].inner_str; This would require the following: traverse_seq_no(&[ 0, // Outer.inners 0, // Vec[0] 1, // Inner.inner_str ])
