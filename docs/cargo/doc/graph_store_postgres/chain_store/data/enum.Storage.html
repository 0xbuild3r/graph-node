pub enum Storage {
    Shared,
    Private(Schema),
}

Storage for a chain. The underlying namespace (database schema) is either public or of the form chain[0-9]+.

Variants

Shared

Chain data is stored in shared tables

Private(Schema)

The chain has its own namespace in the database with dedicated tables
